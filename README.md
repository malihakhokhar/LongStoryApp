# Long Story App

**Long Story App** is an Android application that lets users read, save, and share long stories. The app provides a clean and user-friendly interface for a seamless reading experience.

## Features
- Browse and read long stories across various genres
- Save favorite stories for offline reading
- Share stories with friends
- Simple and intuitive UI
